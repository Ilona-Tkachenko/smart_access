- SmartAccess

SmartAccess – навчальний проєкт, створений для демонстрації роботи системи контролю доступу та обліку відвідуваності. Проєкт подано у вигляді односторінкового сайту, який показує можливості майбутнього програмного продукту.

Опис
Проєкт відображає базову концепцію системи: контроль доступу, облік відвідувачів, статистику та приклад інтерфейсу адміністратора. Сайт є статичним і реалізований на HTML, CSS та JavaScript.

Можливості
- демонстрація принципів контролю доступу
- показ статистики та лічильників
- приклад інтерфейсу системи
- адаптивна верстка
- інформація про технології та сфери застосування

Технології

HTML5  
CSS3  
JavaScript

 Структура
index.html – головна сторінка  
styles.css – стилі  
script.js – логіка  
Smartaccess.png – зображення  
Video.mp4 – велике відео (ігнорується у репозиторії)

Запуск
Відкрити файл index.html у браузері.  
Або запустити через Live Server у VS Code.

Мета

Проєкт створений як демонстраційний MVP системи SmartAccess у рамках навчального завдання.
